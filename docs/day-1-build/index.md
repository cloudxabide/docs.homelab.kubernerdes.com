# Day 1 - Build

Standing up the core infrastructure and clusters:

- Build the admin node (DHCP, DNS, PXE)
- Install Harvester
- Install Rancher Manager
- Add Harvester to Rancher Manager
- Install Observability
- Install the Apps cluster (with Security)

See the source repo's [PLAN.md](https://github.com/cloudxabide/homelab.kubernerdes.com/blob/main/PLAN.md) for the detailed, checkbox-tracked build steps.
